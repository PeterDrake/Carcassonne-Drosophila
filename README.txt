HOW TO USE:
When you start up the game, the console will ask you for player 1(the only player)'s name.
Input the name and hit start. From then on, you will interact with the game via the screen.
Click on a square to place your tile, which is on the bottom left corner of the screen.
If you wish to rotate the tile before placing it, click the rotate button, which will rotate the tile clockwise.
(Graphically, this will not be displayed)
After placing a tile, hit "Add a Follower" button to add a follower.
Have fun!
Accomplish all of your worldly goals.
